# ADR-004: Tiptap for WYSIWYG Editor
**Date:** 2026-05-30 | **Status:** Accepted

## Problem
Choose a rich text editor for blog creation that supports WYSIWYG, markdown, and code blocks.

## Decision
Use Tiptap (headless rich text editor built on ProseMirror).

## Reasons
- Supports WYSIWYG, markdown shortcuts, and code blocks natively
- Headless — full styling control with Tailwind CSS
- Outputs HTML (stored sanitized in DB) or JSON (portable)
- Active maintenance, good React integration
- Extensible for future features (tables, embeds)

## Consequences
- Output must be sanitized server-side before storage (XSS risk — ADR-005)
- Mobile (React Native) will use a different editor (React Native Quill or custom) since Tiptap is web-only
