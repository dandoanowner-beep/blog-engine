# ADR-002: PostgreSQL as Primary Database
**Date:** 2026-05-30 | **Status:** Accepted

## Problem
Choose a database for all platform data.

## Decision
Use PostgreSQL.

## Reasons
- Relational model fits blog data (users, blogs, comments, relationships)
- Native full-text search via tsvector + GIN index (avoids Elasticsearch in Sprint 1–3)
- ACID transactions for friend requests, reactions, reports
- Strong Go driver (pgx) with connection pooling
- Owner's stated preference

## Consequences
- Full-text search may need Elasticsearch at high scale — noted in OPEN_QUESTIONS.md OQ-003
- Must design schema carefully for feed query performance (feed_score denormalization)
